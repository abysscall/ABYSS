# ADR-0011 — AI-Native Account Model

**Status:** Accepted (partially implemented)
**Date:** 2025

## Context

Existing blockchains treat AI agents as an application-layer concern. Each project
that wants an agent model reinvents it: spending limits, permissions, accountability.

This creates inconsistency, security gaps at the application layer, and no shared
standard for what an Agent is allowed to do.

ABYSS treats the AI Agent as a first-class protocol feature, not an application
built on top of generic smart contracts.

## Decision

Every ABYSS account may optionally be paired with a personal AI Agent operating
under a cryptographically enforced `AgentPolicy`:

- **Granted permissions**: what the Agent may do (trade, post, reply, etc.)
- **Spending limit**: maximum AC per transaction or per time window
- **Social limit**: maximum posts/reposts per time window
- **Counterparty restrictions**: which addresses the Agent may interact with

The policy is enforced at the protocol level — not by application logic
that the Agent could circumvent. An Agent that is compromised or misbehaving
cannot exceed the bounds set by the account holder.

**Separation of domains**: financial permissions and social permissions are
governed by separate policy objects. An Agent authorised to post cannot spend
funds, and vice versa, regardless of what instructions it receives.

**Current implementation:**
- `abyss-wallet`: `AgentPermission`, `AgentPolicy`, spending limit enforcement
- `abyss-social`: `AgentSocialPolicy`, `AgentActivityWindow`, rate limit enforcement
- devnet: demonstrates Agent-authorised payment creation and rejection

**Deferred:**
- On-chain Agent marketplace (Phase 5)
- AI inference inside ZK envelopes (Phase 4)
- AI-assisted consensus anomaly detection (Phase 3)

## Consequences

- Users retain full control regardless of Agent behaviour
- Policy enforcement is auditable by anyone reading the code
- Two policy domains require careful design to remain truly independent
- Agent marketplace economics are deferred until the chain is live
